Animations

Recherche & outils

GSAP

GSAP est une bibliothèque d'animation JavaScript standard de GreenSock qui vous permet de créer des animations de haute performance qui fonctionnent dans tous les principaux navigateurs. Se mini site utilise GSAP !
La documentation

AnimeJS

Anime.js ( /ˈæn.ə.meɪ/ ) est une bibliothèque d'animation JavaScript légère avec une API simple, mais puissante. Les exemples sur COS utilise Anime JS !
La documentation

Lottie

Lottie, une solution open source intégrant les animations d'After Effects dans vos applications natives. Conçu par AirBnB en se basant sur les travaux de BodyMovin.
La documentation | Libraire d'animation conçu avec lottie

Mo.js

mo - js est une bibliothèque de motion graphics javascript qui est rapide, prête pour le "retina", modulaire et open source.
La sintax utiliser est différente en comparaison aux outils précédent , mais j'ai étit bluffé par la fluidité des animations conçu avec cettev outils.
La documentation

ThreeJS

Three.js est une bibliothèque JavaScript pour créer des scènes 3D dans un navigateur web. Elle peut être utilisée avec la balise canvas du HTML5 sans avoir besoin d'un plugin.
La documentation

Pixi.js

Pixi.js est une bibliothèque open source permettant de réaliser des animations 2D dans un navigateur web. L'affichage se base sur la technologie WebGL, et la librairie se manipule en JavaScript.
La documentation

Associations de technologies

De plus en plus de développeur, designer et agences utilise le webGl sur leurs sites internet et dans leurs transitions de pages certain , construise pratiquement entièrement le site dans le canvas en webgl. Ces dernières années nous avons aussi vu des sites utiliser les shaders , que ce soit pour des effets à la souris , simuler une texture comme de l'eau, des Sliders d'images mais aussi des transitions de pages. Quand nous commençons à nous intéresser à l'animation dans le web les notions de timeline et d'easings deviennent importantes, l'utilisation de librairie comme GSAP ou AnimeJS une évidence. Un dernier outil qui fonctionne bien que ce soit pour des transitions de pages ou de la micro-interaction c'est Lottie, qui nous permet de récupérer une animation after effect t'elle qu'elle et de le rendre dans le DOM en SVG ou dans un canvas.

La particularité des shaders, c'est que l'effet qu'on créer avec est exportables partout, dans le web, dans la conception de jeux vidéox 3D, la vidéo ou de la retouche photos. Les shaders nous donne le pouvoir de controler l'image et le rendue à notre guise, il est conçu grâce au langage GLSL et le meilleur moyen d'apprendre son utilisation est via la lecture de The Book of Shaders.
